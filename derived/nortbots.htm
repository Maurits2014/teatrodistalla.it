Pagina del componente runtime di FrontPage

Per il corretto funzionamento delle caratteristiche della pagina di destinazione del modulo inviato o del collegamento selezionato è necessario utilizzare un server Web dotato delle estensioni del server di FrontPage.

Per il corretto funzionamento di questo modulo o di altri componenti di FrontPage, è necessario pubblicare il Web corrente in un server che disponga delle estensioni del server di FrontPage.
